This project is a modern, high-quality landing page inspired by a Figma design originally created for Attio CRM.  
The page was fully rebuilt from scratch using Next.js, Tailwind CSS, and Framer Motion, with close attention to layout, spacing, typography, and animation accuracy.

## Overview

The goal of this project was to take a Figma design and faithfully replicate it in code, creating a clean and fully responsive SaaS-style landing page.  
The implementation focuses on pixel consistency, smooth transitions, and a production-ready component structure suitable for real-world marketing websites.

## Tech Stack

- Next.js 15  
- TypeScript  
- Tailwind CSS  
- Framer Motion  
- Lucide React (for icons)

## Features

- Accurate replication of a Figma design  
- Clean and professional SaaS UI  
- Fully responsive layout across all breakpoints  
- Smooth motion effects using Framer Motion  
- Reusable and maintainable component architecture  
- Utility-first styling with Tailwind CSS  
- Fast load times and optimized performance
